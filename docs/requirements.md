## Functional Requirements

- Users can register
- Users can login
- Users can create tasks
- Users can update tasks
- Users can delete tasks
- Users can categorize tasks
- Users can filter tasks by status and priority
- The system can analyze task descriptions and suggest a priority level using AI 
- The system can generate short summaries for tasks using AI
- The system can send automated deadline reminders using n8n

## Non-Functional Requirements

- The API must be secure
- Unauthorized users should not access data
- Input data must be validated
- The system should be scalable
- The API should be well-documented
